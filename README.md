# PDF Chatbot using Gemini API

## Problem
Users struggle to extract useful information from large PDFs.

## Solution
Built an AI-powered chatbot that answers questions from uploaded PDFs using semantic search.

## Tech Stack
Python, Streamlit, LangChain, FAISS, Google Gemini API

## Features
- Upload any PDF
- Ask questions in natural language
- Context-aware answers
- Fast semantic search

## How to Run
1. Clone repo
2. Install requirements
3. Run Streamlit app
